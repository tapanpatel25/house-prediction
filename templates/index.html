About This Project

The House Price Predictor is a machine learning project that estimates house prices based on various features. It uses a linear regression model trained on real estate data to make predictions.

How It Works

Our model takes into account several key factors that influence house prices, including the number of bedrooms and bathrooms, square footage, lot size, age of the house, proximity to the city center, and neighborhood quality. By analyzing these features, we can provide an estimated price for a house.
